OpenRouter demo

This is how AuroraAI chats ‚Äî swap in your key and any model you like.

curl https://openrouter.ai/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_OPENROUTER_KEY" \
  -d '{
    "model": "openrouter/auto",
    "messages": [
      {"role":"system","content":"You are AuroraAI helper."},
      {"role":"user","content":"hello from the vaporwave web"}
    ]
  }'
Tip: Use openrouter/auto or specify any supported model you prefer.
